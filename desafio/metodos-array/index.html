1 find()

Retorna o primeiro elemento que satisfaz a condição. Se nenhum atender, retorna undefined.

Sintaxe

const resultado = array.find((elemento, indice, array) => {
  return condicao; // true = encontrou
});

Casos de uso

  • Buscar um usuário pelo id.
  • Encontrar o primeiro item em estoque.
  • Localizar um produto específico em um carrinho.

Exemplo prático

2 filter()

Retorna um novo array com todos os elementos que satisfazem a condição.

Sintaxe

const novoArray = array.filter((elemento, indice, array) => {
  return condicao; // true = mantém no resultado
});

Casos de uso

  • Filtrar produtos por faixa de preço.
  • Exibir apenas tarefas concluídas.
  • Remover valores inválidos de uma lista.

Exemplo prático

3 map()

Cria um novo array aplicando uma transformação a cada elemento.

Sintaxe

const novoArray = array.map((elemento, indice, array) => {
  return novoValor;
});

Casos de uso

  • Converter preços (aplicar desconto, mudar moeda).
  • Extrair uma propriedade de cada objeto (ex: só os nomes).
  • Formatar dados para exibição.

Exemplo prático

4 reduce()

Reduz o array a um único valor, acumulando resultado a cada iteração.

Sintaxe

const total = array.reduce((acumulador, elemento, indice, array) => {
  return novoAcumulador;
}, valorInicial);

Casos de uso

  • Somar valores (total de um carrinho).
  • Contar ocorrências de itens.
  • Agrupar/transformar uma lista em um objeto.

Exemplo prático

5 forEach()

Executa uma função para cada elemento. Não retorna nada (undefined) — usado por efeito colateral.

Sintaxe

array.forEach((elemento, indice, array) => {
  // executa algo para cada item
});

Casos de uso

  • Imprimir cada item no console.
  • Renderizar elementos na tela.
  • Disparar uma ação por item (salvar, logar).

Exemplo prático

6 some()

Retorna true se pelo menos um elemento satisfizer a condição.

Sintaxe

const existe = array.some((elemento, indice, array) => {
  return condicao;
});

Casos de uso

  • Verificar se existe algum item fora de estoque.
  • Validar se há pelo menos um campo preenchido.
  • Checar permissões (algum usuário é admin?).

Exemplo prático

7 every()

Retorna true somente se todos os elementos satisfizerem a condição.

Sintaxe

const todos = array.every((elemento, indice, array) => {
  return condicao;
});

Casos de uso

  • Validar um formulário (todos os campos válidos?).
  • Checar se todos os itens foram pagos.
  • Garantir que todos os números são positivos.

Exemplo prático

8 includes()

Retorna true se o array contém o valor informado (comparação exata).

Sintaxe

const temItem = array.includes(valor);

Casos de uso

  • Verificar se um item já está no carrinho.
  • Checar se uma tag está em uma lista.
  • Validar se um valor é permitido (lista branca).

Exemplo prático

9 findIndex()

Retorna o índice do primeiro elemento que satisfaz a condição, ou -1 se nenhum atender.

Sintaxe

const indice = array.findIndex((elemento, indice, array) => {
  return condicao;
});

Casos de uso

  • Achar a posição de um item para atualizá-lo.
  • Localizar índice antes de remover com splice().
  • Saber se e onde um item existe.

Exemplo prático

10 sort()

Ordena os elementos do array. Atenção: modifica o array original e ordena como texto por padrão.

Sintaxe

// Alfabética (padrão)
array.sort();

// Numérica / personalizada
array.sort((a, b) => a - b); // crescente

Casos de uso

  • Ordenar uma lista de nomes alfabeticamente.
  • Ordenar produtos por preço.
  • Organizar datas da mais recente à mais antiga.

Exemplo prático
